# FLAMES Calculator

## Overview

The FLAMES calculator is a fun application that determines the relationship between two people based on their names. It uses the popular FLAMES game, which stands for **Friendship, Love, Affection, Marriage, Enemy, and Sister**. This project is designed to provide a light-hearted analysis of relationships in a playful way.

## Features

- Input two names to analyze.
- Calculates the FLAMES score based on the characters in the names.
- Outputs the relationship type based on the FLAMES acronym.

## How It Works

1. **Input**: The user enters two names.
2. **Count Unique Letters**: The calculator counts the unique letters in both names.
3. **FLAMES Logic**: It performs the FLAMES calculation using the counts to determine the relationship.
4. **Output**: Displays the relationship type derived from the FLAMES acronym.
